📉 Principal Component Analysis (PCA)
🔹 What is PCA?
Principal Component Analysis (PCA) is a dimensionality reduction technique used to reduce the number of features while preserving important information.
🔹 Why PCA?
Reduces complexity of data
Removes redundant features
Helps in visualization of high-dimensional data
Improves model performance
🔹 Steps Performed
Standardizing the data
Applying PCA using sklearn.decomposition.PCA
Selecting principal components
Transforming data into lower dimensions
🔹 Outcome
Reduced feature space
Retained maximum variance
Improved interpretability of data


📈 Results & Insights
Identified key trends and relationships in data
Reduced dimensionality without major loss of information
Visualized data in a more interpretable form
